# 接口

| 名称       | 描述                                                  | 类型         | 默认值               |
| ---------- | ----------------------------------------------------- | ------------ | -------------------- |
| dataSource | 树的根节点                                            | `object`     | ---                  |
| scaleRatio | 缩放比例                                              | `number`     | `1`                  |
| options    | 配置项                                                | `object`     | 请参考**配置项**章节 |
| class  | `class`。必须通过 `class` 或 `style` 指定画布的高 | `string`     | ---                  |
| style      | `style`。必须通过 `class` 或 `style` 指定画布的高 | `object`     | ---                  |
